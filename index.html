Mouse Info

Displays Mouse Related Information

Download this project as a .zip file Download this project as a tar.gz file

This is a simple program that displays mouse related information. It shows the current position of the cursor (x and y), the color (in dec and hex) of that position, and the cursor id of the mouse. Copying any piece of data is effortless and can be done by either a button or hotkey. The program has two modes; manual and automatic. When in manual, information updates when you press F4. In automatic, information updates on the fly.
